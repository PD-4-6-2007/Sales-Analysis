# 📊 Sales Analysis Project

## 📌 Overview
This project performs sales data analysis using Python and Jupyter Notebook. The aim is to extract meaningful insights, identify trends, and visualize sales performance using various charts and statistical techniques.

## 🎯 Objectives
- Analyze sales data efficiently.
- Identify sales trends and patterns.
- Calculate key metrics such as total sales, average sales, and quantity sold.
- Visualize data using charts and graphs.
- Generate insights for better business decisions.

## 📊 Dataset Information

The dataset contains the following columns:

| Column Name | Description |
|--------------|-------------|
| Product ID | Unique product identifier |
| Sales Date | Date of sale |
| Sales Rep | Sales representative |
| Region | North, South, East, West |
| Sales Amount | Total amount of sales |
| Quantity Sold | Number of units sold |
| Product Category | Category of the product |
| Unit Cost | Cost per unit |
| Unit Price | Selling price per unit |
| Customer Type | Customer classification |
